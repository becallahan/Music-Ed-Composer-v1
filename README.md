# MusicEd Composer

## Mission

Help students understand the relationship between piano-roll composition and standard notation.

**Core educational outcome:**

> "That square became that note."

MusicEd Composer is a visual composition environment that bridges grid-based composition, playback, and traditional music notation.

---

## Version 1 Goals

### Composer Grid

- 8 measures
- 4/4 time
- 64 eighth-note positions
- Diatonic pitch range from Middle C to C5
- Touch-friendly input
- Melody mode
- Harmony mode

### Playback

- Play
- Stop
- Loop
- Tempo control

### Sounds

- Piano
- Bass
- Bell
- Synth
- Drum
- Weird/Fun
- Sine
- Triangle
- Saw
- Square

### Notation

- Real-time staff display
- Treble clef
- Correct staff placement
- Rhythm grouping
- Automatic rests
- Visual connection between grid actions and notation

---

## Music Nerd

Future AI-powered harmony analysis.

When students create a chord, Music Nerd can explain:

- Chord names
- Roman numerals
- Jazz symbols
- Figured bass
- Harmonic function
- Tension and resolution

At beginner, intermediate, and advanced levels.

## Visual Design

### Color Palette

| Purpose | Color |
|----------|----------|
| Background | #F2F2F0 |
| Panels | #FFFFFF |
| Text | #1C1C1C |
| Primary Accent | #B23A2B |
| Mustard Highlight | #D4A017 |
| Success Green | #2E8B57 |
| Measure Lines | #999999 |
| Staff Notation | #000000 |

### Design Principle

Grid notes may appear in red.

Standard notation remains black on white.

This reinforces the learning objective:

> The red square became the black note.

---

## Educational Principle

Every action in the grid should immediately translate into notation.

Students should be able to:

1. Click a square
2. Hear a sound
3. See notation appear
4. Understand the connection

The software teaches notation through composition.
